Controls:

WASD / Arrowkeys: Vertical and horizontal movement of the player's avatar
SPACE: Activate the deflector shield
SHIFT: Activate the timewarp feature
ESC: Open the pause menu ingame

WASD / Pfeiltasten: Vertikale und horizontale Steuerung des Spieler-Avatars
SPACE: Aktivierung des Deflektorschilds
SHIFT: Aktivierung des Zeitkrümmers
ESC: Öffnen des Pausemenüs während dem Spielen

Installation: Download node.js and Typescript for Visual Studio Code, as well as the Extension Live Server.
Start the Application through Live Server in VSC and play the game in your Browser window.

Installation: Downloade node.js und Typescript für Visual Studio Code, sowie die Erweiterung Live Server.
Starte die Anwendung über Live Server in VSC und spiele das Spiel im Browserfenster.
